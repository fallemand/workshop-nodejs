# Workshop Node! 

[![Node](https://nodejs.vn/uploads/system/touchicon-48.png)]()

Para ejecutar el proyecto, posicionarse en la carpeta /tp:

* Build assets with
    ```npm run build```

* Run with
    ```npm run watch```
    ```npm run start-dev``` 
    ```npm run start``` 

* Run test with
    ```npm run test-unit```
    ```npm run setup-e2e```
    ```npm run start-test```
    ```npm run test-e2e```
