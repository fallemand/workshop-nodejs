## Develop
`Node: v6.11.0`  
### 1) Install dependencies:
```
npm i
```
### 2) Compile  assets
```
npm run build
```
or
```
npm run watch
```
### 3) Start app:
```
npm run start-dev
```
### 4) Make sure to have this added to your etc/host file:
```
127.0.0.1	localhost
```
### 5) Navigate to:
```
http://localhost:3000/
```

## Run Tests
### Unit tests
```
npm run test:unit
```
### Fuctional tests
```
npm run test:e2e
```
