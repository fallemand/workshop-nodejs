# Development



## To run application locally

### Commands
`npm install`

`npm run build`

### To run App with development env
`npm run start:dev`


### To run App with production env
`npm run start`

`URL: http://localhost:3000/app

# Tests

## To run unit tests
`npm run test:unit`

## To run functional tests
`npm run test:e2e`
